# 1.2 — Docs referenciam classes inexistentes (CRÍTICO)

## Problema
O commit `9953db9` renomeou as classes para o prefixo `SpringSessionLite*`/`SpringLite*`,
mas `docs/01-instalacao-e-uso.md` e `docs/03-como-funciona.md` ainda citam nomes antigos:
`SessionLiteService`, `SessionUser`, `CurrentSession`, `Session`, `SessionRepository`,
`UserSessionLiteService`, `SessionAuthenticationFilter`. Doc 01 instala `1.0.0` (pom = `1.0.1`).

Nomes reais:
| Doc (errado) | Real |
|--------------|------|
| `SessionLiteService` | `SpringSessionLiteService` |
| `SessionUser` | `SpringSessionLiteUser` |
| `@CurrentSession` | `@SpringSessionLiteCurrentSession` |
| `Session` | `SpringLiteSession` |
| `SessionRepository` | `SpringLiteSessionRepository` |
| `UserSessionLiteService` | `SpringSessionLiteUserService` |
| `SessionAuthenticationFilter` | `SpringSessionLiteAuthenticationFilter` |

## Impacto
Copy-paste das docs não compila. Onboarding quebrado.
